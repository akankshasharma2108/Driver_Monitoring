# Driver Alert System (Eye Closure Detection)

## Project Overview
The **Driver Alert System** is a real-time computer vision project that monitors a driver using a webcam. It detects whether the driver’s eyes are open or closed and provides a live alert.

This system helps in identifying **driver drowsiness**, which is one of the major causes of road accidents. The project is built using **Python, OpenCV, and Streamlit**, making it simple, interactive, and easy to run.

---

## Objective
- Detect driver fatigue using eye closure detection  
- Provide real-time alert messages  
- Build a practical AI-based safety system  
- Create a beginner-friendly real-world project  

---

## Features
- Real-time eye detection using webcam  
- Detects closed eyes and shows alert  
- Displays driver status (Safe / Not Safe)  
- Runs on browser using Streamlit  
- Lightweight and easy to use  
- Beginner-friendly implementation  

---

## Technologies Used
- **Python 3.13**  
- **OpenCV** – Computer Vision  
- **Streamlit** – Web Interface  
- **NumPy** – Numerical operations  
- **Imutils** – Video handling
